# nextfounders.pl

Strona NextFounders — niezależnej organizacji zrzeszającej młodych founderów i founderki, którzy faktycznie działają. Wrocław.

Organizacja działa w ramach Next Real Sp. z o.o.

## Stack

- Next.js (App Router) + React
- Tailwind CSS v4
- TypeScript

## Rozwój lokalny

```bash
npm install
npm run dev
```

Strona dostępna pod [http://localhost:3000](http://localhost:3000).

## Build produkcyjny

```bash
npm run build
```

## Deploy

Projekt jest przygotowany pod Vercel — wystarczy podpiąć repozytorium i domenę `nextfounders.pl`.
